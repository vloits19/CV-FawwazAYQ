Profil

Saat ini saya adalah siswa kelas XI di SMK Negeri 1 Bawang Banjarnegara, jurusan Pengembangan Perangkat Lunak dan Gim. Saya sedang mempelajari berbagai aspek pengembangan game termasuk pemrograman, desain game, dan aset digital sebagai persiapan untuk karir di industri game.

Tahun 2024-2025 merupakan tahun penting bagi saya untuk menyelesaikan pendidikan sekaligus mengembangkan portofolio proyek-proyek game. Saya sangat tertarik dengan pengembangan game Unity dan terus mengasah kemampuan pemrograman C# serta Unity untuk mewujudkan impian menjadi game developer profesional. Saat ini saya sedang mengembangkan game Flappy Bird untuk berbagai platform termasuk web dan mobile.
